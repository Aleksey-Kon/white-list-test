# 📱 Проверка белых списков

Мобильное приложение для проверки белых списков, разработанное на **React Native** с использованием **Expo**.

![Скриншот приложения](./.assets/screenshot.png)

## 🚀 Возможности

- ⚡Проверка белых списков по доменам
- 🎨 Современный интерфейс с навигацией
- ⚡ Оптимизированная производительность

## 🛠 Технологии

- **React Native** 0.81.5
- **Expo** ~54.0.33
- **Expo Router** ~6.0.23
- **React** 19.1.0
- **TypeScript** ~5.9.2
- **React Navigation** для навигации
- **React Native Reanimated** для анимаций

## 📦 Установка

1. Клонируйте репозиторий:

```bash
git clone <URL_РЕПОЗИТОРИЯ>
cd testwhitelist
```

2. Установите зависимости:

```bash
npm install
```

3. Запустите приложение:

```bash
npx expo start
```

## 📱 Запуск на устройствах

### Android

```bash
npm run android
```

### iOS

```bash
npm run ios
```

### Web

```bash
npm run web
```
